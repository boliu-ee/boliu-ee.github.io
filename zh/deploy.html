部署与自定义域名

这一页给你留作真正上线时的操作说明。现在这套站点已经是 GitHub Pages / Gitee Pages 兼容的纯静态结构。

GitHub Pages

  1. 新建仓库：yourname.github.io
  2. 把本站点所有文件上传到仓库根目录
  3. Settings → Pages → Deploy from branch → main / root
  4. 等待生成站点地址

自定义域名

在仓库根目录放置 CNAME 文件，例如：

www.yourdomain.com

然后在域名解析后台配置 CNAME 或 A 记录。

Gitee Pages

  1. 创建公开仓库并上传同一份静态文件
  2. 开启 Gitee Pages 并手动启动部署
  3. 建议作为国内镜像站，而 GitHub Pages 作为主站

我建议你这样做

  • 主站：GitHub Pages
  • 镜像：Gitee Pages
  • 域名：绑定到 GitHub Pages
